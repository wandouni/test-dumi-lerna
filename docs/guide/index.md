## 什么是 dumi？

dumi，中文发音嘟米，是一款为组件开发场景而生的文档工具，与 father 一起为开发者提供一站式的组件开发体验，father 负责构建，而 dumi 负责组件开发及组件文档生成。

## 特性

- 开箱即用，将注意力集中在组件开发和文档编写上
- 丰富的 Markdown 扩展，不止于渲染组件 demo

## 快速上手

### 环境准备

首先得有 node，并确保 node 版本是 10.13 或以上。

```
$ node -v
v10.13.0
```
